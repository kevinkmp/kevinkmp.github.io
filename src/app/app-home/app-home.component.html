SwiftIQ Sample Shell App

I created this app after reading a bunch of articles and tutorials for Angular2. I wanted to see if it was actually as easy as some of the tutorials made it seem. For the most part it was. There were a few things that took a bit of researching to figure out, most of the time this was due to the articles and videos dealing with now outdated beta versions of Angular2.

This sample app was created using the angular-cli tool inside of WebStorm. Boy does this ever help to speed things up!

Features

  1. Navigation Components - I created two components for navigation, nav-primary and nav-secondary.
  2. Class Inheritance - I created two base classes, BaseApplication and BaseApplicationResult. Some of the components in this application extend these base classes.
  3. Nested Components - The "Views" application page has a nested component which acts to show results.
  4. angular-cli - This tool (which you can run from the terminal inside of WebStorm) makes creating components as easy as typing "ng generate component my-sample". Still too long? "ng g c my-sample". This creates a folder "my-sample", and inside of the folder it makes four files:
    my-sample.component.css
    my-sample.component.html
    my-sample.component.spec.ts (automatically generated unit tests)
    my-sample.component.ts (TypeScript file for the component)
